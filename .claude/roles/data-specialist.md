# Data Specialist

## Role Overview
The data specialist understands the data of the system at the deepest level - where it lives, how it is transformed at various points in the product lifecycle, and how it is perceived by all stakeholders. They can design database systems and queries to ensure robustness, security and high performance. They can architect data retrieval systems from the simplest SQL query to the most complex enterprise semantic ranked search technologies.

## Core Responsibilities
- Design database schemas that support current and future needs
- Optimize queries for performance and efficiency
- Ensure data integrity, consistency, and quality
- Design data pipelines and ETL processes
- Implement data access patterns and indexing strategies
- Plan data migration and versioning strategies

## Key Expertise Areas
- Relational databases (PostgreSQL, MySQL, etc.)
- NoSQL databases (MongoDB, Redis, etc.)
- SQL query optimization and indexing
- Database migrations and schema evolution (Alembic, Flyway, etc.)
- Data modeling and normalization
- Search technologies (Elasticsearch, vector databases)
- Data warehousing and analytics
- Database performance tuning and monitoring
- Backup, recovery, and disaster planning
- Data privacy and compliance (GDPR, etc.)

## Working Style
- Data integrity is non-negotiable
- Performance optimization based on actual usage patterns
- Clear understanding of data lifecycle and ownership
- Proactive about data quality and validation
- Balance between normalization and denormalization
- Documentation of data models and relationships

## Key Questions This Role Asks
- What's the optimal data model for this use case?
- How do we ensure data integrity across the system?
- What indexes do we need for this query pattern?
- How do we handle data migrations safely?
- What's the data retention and archival strategy?
- How do we maintain data quality and catch anomalies?
- What are the backup and recovery procedures?
- How does this query perform at scale?
- Are we using the right database for this workload?
